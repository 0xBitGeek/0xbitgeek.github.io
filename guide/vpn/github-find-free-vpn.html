利用Github实现免费科学上网

前言

由于众所周知的原因，在国内访问一些网站比较困难，所以给大家提供一个免费科学上网的方法，希望可以帮助各位自由冲浪。

声明：本文仅作为技术分享，使用者请务必遵守法律法规。

以下几个问题可能是你先需要知道的。

1 适用人群

这些免费节点适用于偶尔有科学上网需求，但感觉不值得付费的轻度用户，如查文献，查资料或者防失联等使用场景，速度尚可，但是延迟可能比较高。

如果你经常使用 Twitter，Discord，Gmail 等账户，一个稳定的网络环境就十分必要了，请参阅：

科学上网如何挑选付费“机场”

2 为什么会出现免费节点？

出于推广收费节点，吸粉等目的，有些人会拿出一部分资源开放免费使用。所以就出现了一些汇总这些免费节点的项目。对于页面内的付费和引流内容请不要轻易相信。

3 安全吗？

不安全，但也相对安全。即使是付费的也不安全，因为所有的请求数据会经过别人的服务器并被记录。但现在 https 基本普及，请求的内容无法被得知。并且由于经常会切换节点，所以追踪也增加了难度。即使是自己搭建 VPS，由于是固定 IP，也很容易被 GFW 特征定位到而被封禁。

不过在使用免费节点时还是要注意，配置绕过大陆地址，用完及时关闭等。

4 我知道 XX 加速器，某灯等软件可以一键科学上网，你这个也太麻烦了吧？

我个人非常反对使用封装好的一键科学上网软件或 App，第一是节点延迟和速度无法保证，且使用人数众多（这也是造成一些海外账户封号的主要原因），第二，非开源客户端可能存在安全隐患或后门，尤其是 Android 和 Windows；第三，容易被钓鱼执法；第四，付费跑路风险非常之高等等。

好了接下来进入正题。

搜索免费节点

首先确认你的网络环境是否可以正常访问 github.com 以及下载 Github 中的文件，经过实测下列软件的项目库地址都被屏蔽了，最好先查看以下解决方法：

GitHub 访问慢打不开？快使用加速神器 FastGithub

在 GitHub 中搜索关键字如“免费 节点”，右上角可以选择不同的排序方式筛选结果，推荐选择 Recently updated（最近更新） 可以筛选出活跃项目，这些都是收集了全网免费节点资源的项目，基本每天都会更新。

比如我挑选出来如下几个我测试时可正常使用的，如失效请请自行寻找。

  • https://github.com/freefq/free
  • https://github.com/yu-steven/openit
  • https://github.com/aiboboxx/v2rayfree
  • https://github.com/alanbobs999/TopFreeProxies

打开其中一个项目会看到类似上图这样的订阅链接，复制订阅链接，页面内容可能会有不一致请自行查看描述。

Base64 可供 V2Ray 使用，Clash 都是专用链接。接下来教你各个客户端如何使用。

冲浪客户端下载和使用

目前主流的客户端有 V2Ray 和 Clash 两种，v2Ray 客户端无法使用 Clash 节点订阅，Clash 要使用 V2Ray 订阅也需要进行转换，不适合新手操作，所以我们直接使用现成的 订阅链接 就好，v2Ray 和 Clash 两者选其一就可以了。

请根据你所使用的设备类型选择客户端，跳转至对应章节查看新手教程。

iOS

因为 iOS 较为特殊所以单独说一下。 因为众所周知的原因科学上网工具类软件无法在国区 App Store 上架。所以你需要先注册一个 海外 Apple ID 。如何注册请参阅：

iOS 注册海外 Apple ID 最新教程

由于注册过程中可能先需要科学上网，容易陷入一个死循环，因为不能科学上网所以需要注册；因为注册不了所以没法科学上网……所以我推荐你先在 Windows、Android、macOS 等设备上配置好科学上网，然后共享代理给 iOS 连接后再进行注册。请参阅：

局域网内多台设备如何共享代理

如果你想白嫖到底，那么我推荐你使用 V2Ray 客户端 OneClick，它是 免费 的，但体验一般。

如果你想使用 Shadowrocket，售价 2.99 美元 （美区）。请自行在 某宝 搜索关键词 iOS 礼品卡 进行充值后购买。
